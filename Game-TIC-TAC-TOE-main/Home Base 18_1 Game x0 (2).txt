Tic-Tac-Toe Game
Write a program that allows two players to play a game of tic-tac-toe. Use a twodimensional char array with three rows and three columns as the game board. Each element of the array should be initialized with an asterisk (*). The program should run a loop that * Displays the contents of the board array 
* Allows player 1 to select a location on the board for an X. The program should ask the user to enter the row and column number.
* Allows player 2 to select a location on the board for an O. The program should ask the user to enter the row and column number.
* Determines whether a player has won, or a tie has occurred. If a player has won, the program should declare that player the winner and end. If a tie has occurred, the program should say so and end. Player 1 wins when there are three Xs in a row on the game board. The Xs can appear in a row, in a column, or diagonally across the board. A tie occurs when all of the locations on the board are full, but there is no winner.

Гра в хрестики-нулики
Напишіть програму, яка дозволяє двом гравцям грати в гру в хрестики-нолики. 
Використовуйте двовимірний char- масив з трьома рядками та трьома стовпцями як ігрове поле. 
Кожен елемент масиву слід ініціалізувати зірочкою (*) чи іншим символом. 
Програма повинна запускати цикл, що
	* Відображає вміст масиву дошки
	* Дозволяє гравцеві 1 вибрати місце на дошці для X. Програма повинна запитувати користувача ввести номер рядка та стовпця.
	* Дозволяє гравцеві 2 вибрати місце на дошці для О. Програма повинна запитувати користувача ввести номер рядка та стовпця.
	* Визначає, виграв гравець чи відбулася нічия. 
Якщо гравець виграв, програма повинна оголосити цього гравця переможцем і закінчити. Якщо нічия, то програма повинна повідомим про це і завершити свою роботу.
Гравець 1 виграє, коли на ігровій дошці є три Х поспіль. Можуть з’являтися символи X в ряд, у стовпець або по діагоналі по всій дошці. 
Нічия  відбувається, коли всі місця на дошці заповнені, але переможця немає.
