1. Визначити рекурсивну функцію обчислення x у степені n  для дробового x (x ≠ 0) та цілого n.
	double power(double x, int n)

2. Визначити рекурсивну функцію обчислення n – го числа Фібоначі  unsigned int fib(unsigned number)
Перших  два числа Фібоначі рівні 0 та 1, кожне  наступне обчислюється як сума двох попередніх. 	
f0 = 0; f1 = 1; 
f(j) = f(j-1)+ f(j-2);  j =  2, 3,.…	
За допомогою функції обчислити N чисел Фібоначі.

Виконати на вибір одне із  завдань
3.1. Написати рекурсивну функцію(або шаблон), яка отримує масив і повертає найбільший його елемент.

3.2. Написати рекурсивну логічну функцію(або  шаблон), яка перевіряє чи переданий у функцію масив 
впорядкований за зростанням(не спаданням).

3.3 Написати рекурсивну функцію(або  шаблон), яка отримує одновимірний масив  та обертає його
(тут зручно  передавати лівий та правий індекси у масиві для реверсу).



Додаткові завдання
___________________________________________________________
Дана шахівниця розміром 8х8 і шаховий кінь. Програма має вимагати від користувача координати
клітини поля й поставити туди коня. 
Завдання програми знайти і вивести шлях коня, при якому він обійде всі клітини дошки, стаючи в кожну клітину тільки один раз.
(Оскільки процес пошуку шляху для різних початкових клітин може затягнутися, то рекомендується спочатку випробувати завдання на полі розміром 6х6).
У програмі необхідно використовувати рекурсію.
